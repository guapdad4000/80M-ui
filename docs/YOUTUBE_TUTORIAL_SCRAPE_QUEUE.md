# YouTube Tutorial Scrape Queue for 80M Motion Lessons

This file is the production intake for future branded infographic videos. The goal is not to copy other creators. The goal is to collect tutorial topics, extract transcript-level learning points where permitted, rewrite them in 80M language, and hand structured briefs to a motion agent for original 80M-branded explainers.

## Scrape Rules

- Prefer official docs and official product channels when available.
- Use YouTube transcripts and metadata as research input only.
- Do not reuse video clips, thumbnails, music, creator graphics, or exact script language without permission.
- Every generated video should be original: 80M voice, 80M visuals, 80M examples, and beginner-safe explanations.
- Record source URL, channel, duration, intended course, target lesson, scrape status, and motion angle.
